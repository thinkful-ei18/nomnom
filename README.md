# For the love of food

For the love of food is recipe sharing application. Users can create, search, and share recipes with an easy to use interface.

## Deployed link

<http://epic-albattani-61c20d.netlify.com/>

## Tech Stack

### Client

* React
* React Router
* Redux
* Redux Thunk
* Redux Form
* Sass

### Server

* BCrypt
* Express
* JSON Web Tokens
* Mongoose
* Passport

### Database

* MongoDB
